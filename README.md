# Temple Run - Browser Game

A fun Temple Run-style endless runner game built with HTML5 Canvas and JavaScript!

## How to Play

1. **Open `index.html`** in your web browser
2. Use arrow keys to control the character:
   - **↑ Arrow Up** - Jump over obstacles
   - **↓ Arrow Down** - Slide under obstacles
   - **← Arrow Left** - Move to left lane
   - **→ Arrow Right** - Move to right lane

## Game Features

- **Three lanes** - Navigate between left, center, and right lanes
- **Obstacles** - Avoid walls and spike traps
- **Coins** - Collect coins for bonus points
- **Progressive difficulty** - Game gets faster and obstacles spawn more frequently
- **Score tracking** - Earn points by surviving and collecting coins
- **Smooth animations** - Fluid running, jumping, and sliding mechanics

## Controls

| Key | Action |
|-----|--------|
| ↑ | Jump |
| ↓ | Slide |
| ← | Move Left |
| → | Move Right |

## How to Run

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Start playing!

No installation or server needed - runs entirely in your browser!

## Game Mechanics

- **Jump** over obstacles in your current lane
- **Slide** to pass under walls without jumping
- **Move lanes** to avoid oncoming threats
- **Collect coins** for extra points (+10 points each)
- **Stay alive** as long as possible

## Tips

- Slide under walls instead of jumping for a faster escape
- Plan your lane changes ahead of time
- Don't jump too early - time your jumps carefully
- Collect coins for bonus points
- The game gets harder the longer you survive!

Enjoy the game! 🎮
